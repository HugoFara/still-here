# Species photo credits

These are **reference photographs of each species**, shown so a follower can
recognise the animal at a glance. They are **illustrative of the species — not
of the individual tracked bird**, and are unrelated to the Movebank tracking
studies.

Each is a freely-licensed image from [Wikimedia Commons](https://commons.wikimedia.org),
**vendored locally** (downloaded at 600 px, no hot-linking). Licenses are
surfaced in the UI (the follow view's provenance block) and respected here.
CC BY / CC BY-SA require attribution; the credit lines below satisfy that.

The Red Kite photograph was taken in Switzerland — fitting, since the roster's
Red Kite (*Aare*) is a Swiss-plateau bird.

To change or refresh an image, replace the file here, update the credit in this
table **and** in `SPECIES_PHOTO` in `src/web/app.js`, then re-run `npm run static`.
